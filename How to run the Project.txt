How to run the Project

1. After download the zip file, Extract the file and copy the folder 'myCMS'.
2. Paste inside root directory of XAMPP (C:\xampp\htdocs).
3. Now open XAMPP and Start Apache & MySQL
4. Click admin in MySQL or go to [http://localhost/phpmyadmin/] .
5. Create database name 'myCMS'.
6. Import 'mycms.sql' file(given inside the zip package in SQL file folder)
7. now go to [http://localhost/myCMS/Blog.php] (Client Side)
8. for Server Side: click 'Server Side' in the Navigation Bar.
